# Introduction

Welcome! This documentation covers the entire Hana Translate suite, which consists of:

- **Hana Translate Client Library**: A developer-friendly library for integrating translation features into your applications.
- **Remote Translation Server & Management**: A robust backend service providing translation APIs and management tools, fully compatible with the client library.

## Overview

The Hana Translate suite enables seamless integration of translation services into your workflow, whether you are building client applications, managing translations remotely, or tracking progress via a dashboard. This documentation will help you get started, understand each component, and find resources for further development.

### Key Features

- **Unified Translation Ecosystem:** Client library, server, and dashboard work together for a complete solution.
- **Developer Friendly:** Easy-to-use client library for rapid integration.
- **Centralized Management:** Remote server offers scalable translation management and API endpoints.
