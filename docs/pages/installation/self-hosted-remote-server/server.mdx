# Self-hosted Remote Server


## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) (latest version)
- PostgreSQL database
- Redis server

### Environment Setup

clone `./server` from `https://github.com/FINGU-GRINDA/runfix-container.git`

Copy the example environment file and configure your settings:

```bash
cp .env.example .env
```

Update the following variables in your `.env` following `.env.example`:

### Installation

```bash
# Install dependencies
bun install

# Run database migrations
bun prisma migrate dev
```

### Running the Server

#### Development Mode

```bash
bun run dev
```

#### Production Mode

```bash
bun run start
```

### Docker Deployment

You can also run the server using Docker:

```bash
# Build and start the containers
docker-compose up -d
```

server is up at `http://localhost:8000`

documentation is available at `http://localhost:8000/docs`
